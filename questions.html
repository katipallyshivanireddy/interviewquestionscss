What is the difference between the visibility:hidden and display:none?

visibilty:hidden:- When an element is set to be visibility:hidden,the element is still rendered on the page, but it is not visible.
- The element is hidden from view, but the space it occupies is preserved, so it still affects the layout of the page
-It hides the element without changing the layout of other elements around it.

display:none:- When an element is set to display :none ,the element is removed from the normal document flow,and it is not rendered on the page at all.
-The element is not visible and does not take up any space in the layout.

What is position property?

How many types of positions values we can give ?

In css you can specify different position values to control the positioning of an element on a webpage.The main position values in css are:
1.static:- this is the default position value where elements are positioned according to the normal flow of the element
2.relative:- It allows you to adjust the element's position using top, right, bottom, and left
3.absolute:- REmoves the element from the normal document flow and positions it relative to its closet positioned ancestor
4.fixed:- Position the element relative to the the viewport,so it stays fixed in its position even when the page is scrolled
5.sticky:- Acts like a combination of relative and fixed positioning .The elemnt is treated as relative positioning until it reaches a specified scroll position, then it is treated as fixed positioning

What is the difference between the position:sticky and position:fixed?

The difference betweenthe position:sticky and position:fixed is that with the position sticky ,the element is positioned based on the users scroll position.It toggles between relative and fixed positioning as you scroll.on the other hand, position fixed keeps the element fixed in a specific position on the screen,regardless of scrolling.

what is the difference between the position -relative and position-absolute?

position:absolute:-the element can move anywhere on the page,and it doesn't affect the position of other elements
